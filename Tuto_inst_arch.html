STM32 DD112 Wattmeter

Importer le programme depuis l'archive du projet

Logiciel nécessaire

Il faut au préalable avoir installé la suite logiciel ST suivante :

  • STM32CubeMX
  • STM32CubeIDE
  • STM32CubeProgrammer

Importer le projet

On démarre STM32CubeIDE :

  1. On clique sur le i
  2. On clique sur Import project

Ensuite, on clique sur Archive... :

On choisi l'archive du projet présente dans le dossier Projet STM32 DD112 Wattmeter à coté du Readme

On clique maintenant sur Finish

Après quelques instant, quand le programme a fini l'importation :

  1. On déroule le projet et sélectionne STM32_DD112_Wattmeter
  2. Puis on compile le projet
  3. On vérifie qu'il n'y a pas d'erreur
  4. On branche la carte si c'est pas déjà fait et on téléverse le programme

Si tout est correct vous devriez avoir l'écran qui affiche l'init puis les datas.
